LINE BOT DESCRIPTION

AutoPurchaserは、Lineをインターフェースとして動作します。Line Botに定型文を送信することによって商品の監視、購入を実行します。 このページでは、Autopurchaserで使用可能な定型文についての説明が書かれています。

商品監視メッセージ

AutoPurchaserで商品を監視するためのメッセージについて記載しています。

Please see below for the details

監視をする商品のURLを入力してください。現在はAmazonのみに対応しており、URLはhttps://www.amazon.co.jpから始まる必要があります。

監視の上限金額を入力してください。maxprice_4000と入力した場合、監視のタイミングで監視商品が4001円以上なら購入は行われず、4000円以下であれば自動購入が実行されます。

XXXXの部分には秒数を入力してください。schedule_30と入力すると30秒間隔で商品を監視します。

スケジューラを終了します。
